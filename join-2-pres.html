Strategy 1: Policies, Fees & Incentives

Implementation Cost: LOW

Actions Under Consideration:

  • Fee Waivers for Affordable Units (Both jurisdictions)
  • ADU Ordinance (Town lead, County follow)
  • STR Caps or Bans (County primary, Town coordinate)
  • Density Bonuses for Affordable Units (Both jurisdictions)

Decisions for This Strategy:

Which actions to pursue • Who leads each action • Priority order

Strategy 2: Optimize Existing Stock

Implementation Cost: LOW-MODERATE

Actions Under Consideration:

  • Anti-Displacement Policy/Rules (Joint development)
  • Acquisition and Rehabilitation (Both fund, shared implementation)
  • Right of First Refusal Programs (Each entity for own properties)

Decisions for This Strategy:

Which actions to pursue • Who leads each action • Priority order

Strategy 3: Strategic Development

Implementation Cost: HIGH

Actions Under Consideration:

  • Land Acquisition/Banking (Both identify sites)
  • Secure Options or Purchase Agreements (As opportunities arise)
  • Shared Housing Staff Position (Joint position, shared costs)
  • Employer Partnerships (Both cultivate)
  • Legislative Advocacy (Joint efforts)
  • Housing Development (Joint efforts)

Decisions for This Strategy:

Which actions to pursue • Who leads each action • Priority order

Strategy 4: Sustainable Revenue

Implementation Cost: VARIABLE

Actions Under Consideration:

  • Property Tax Allocation (Each entity for own mill levy)
  • Housing Impact/Linkage Fees (Both implement separately)
  • Inclusionary Zoning with Fee in Lieu (Town pilot, County evaluate)
  • Partnership Leverage & Community Fundraising (Joint effort)

Decisions for This Strategy:

Which actions to pursue • Who leads each action • Priority order
